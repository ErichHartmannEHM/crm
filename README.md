# CARD Wallet (обновлённый)

## Установка
1. Создайте БД MySQL 8+ и обновите `config.php` по необходимости.
2. Выполните SQL из `_schema_full.sql` (создаст таблицы и админа: `admin@example.com` / `admin123`).
3. Задайте переменные окружения (опционально):
   - `TG_BOT_TOKEN` — токен Telegram бота
   - `TG_ADMIN_CHAT_ID` — чат ID админа
4. Разместите проект в `service-ref.sbs/public_html`.
5. Войдите: `/login.php`.

## Роли и логика
- Команды → Баеры → Карты.
- Карты имеют статусы: waiting/in_work/processing/await_refund/archived.
- Операции: пополнение/списание/холд (все суммы в UAH).
- Лимит карты: 100 000 UAH на месяц (пересчёт по операциям текущего месяца).
- Баланс команды — в USD; при списании с карты уменьшается баланс команды в USD по текущему курсу.
- Уведомления Telegram: баеру — по операциям; админу — при порогах.

## Директории
- Загрузки выписок: `storage/uploads/statements` (создаётся автоматически).

## Безопасность
- PAN/CVV хранятся в шифрованном виде, PAN выводится маской, CVV не отображается.
- Формы защищены CSRF.
